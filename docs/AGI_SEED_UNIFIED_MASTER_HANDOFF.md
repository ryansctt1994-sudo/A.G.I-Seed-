## 1. Executive Overview

This document defines the current constitutional baseline for the AGI Seed research portfolio.

The portfolio is no longer treated as a collection of unrelated AI experiments. It is now organized as a vertically integrated governance architecture whose purpose is to safely transform observations into externally validated evidence through reproducible engineering.

The portfolio intentionally separates:

- observation from measurement
- measurement from governance
- governance from constitutional interpretation
- architecture from evidence
- reasoning from authority
- implementation from validation

The engineering embodiment of these principles is **AGI Seed**, a governed execution system that constrains untrusted model outputs through layered verification before any execution occurs.

The central research question is not:

> Can an AI autonomously write software?

The central research question is:

> Can an untrusted model improve software through bounded, reproducible, receipt-governed feedback while never controlling execution machinery or promoting its own authority?

Everything in the portfolio exists to answer that question scientifically.

## 3. Constitutional Principles

The following principles are frozen as the current constitutional baseline.

### Invariant 1: Reasoning Never Grants Authority

Reasoning is advisory. Authority is earned only through evidence.

### Invariant 2: Sophistication Never Raises Evidence

Architectural refinement, conceptual elegance, or symbolic sophistication cannot raise an evidence level.

Only the following may promote evidence:

- execution
- independent reproduction
- adversarial or external review
- operational deployment

### Invariant 3: Execution Machinery Is Independent from Model Intent

Models propose. Deterministic systems decide.

### Invariant 4: Every State Transition Must Be Observable

No invisible promotion. No silent authority escalation. No unrecorded execution boundary crossing.

### Invariant 5: Sufficiency Precedes Verification

Verification is inadmissible unless evaluator capacity is sufficient to judge the claim.

### Invariant 6: Evidence Promotion Requires Receipts

No receipt means no promotion.

### Invariant 7: Governance Must Itself Be Reproducible

Governance decisions must satisfy the same replay and audit standards imposed on model behavior.

## 7. Sufficiency Governance

The portfolio adopts the Sufficiency Lock as a mandatory pre-verification gate.

Verification does not begin by asking:

> Can this be tested?

It begins by asking:

> Are we capable of judging this test correctly?

Formal metric:

```text
SR = E / G
```

Where:

- **E** = available evaluator capacity or evidence capacity
- **G** = governance demand or judgment difficulty

Verification may proceed only when:

```text
SR >= 1
```

This prevents governance theater: evaluations that appear rigorous but cannot actually judge the claim being tested.
